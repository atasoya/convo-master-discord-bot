# All Commands

## Commands

```
- c.help
- c.general [language]
- c.question [language]
- c.personal [language]
- c.thisorthat [language]
- c.wouldyourather [language]
- c.translate [language] [sentence]
- c.movie
- c.languages
- c.add
```

## Information

The bot is available in multiple languages. To see the list of all of them type `c.languages`.  
The default language is english (en).
